# NUTRILOG

NUTRILOG is a simple nutrition and weight tracking application with four main sections:

## Food Database
In this tab, you can add food items and define their nutritional values per 100 grams: calories, protein, fats, and carbohydrates.

## Diary
In the Diary tab, you can select foods from your Food Database and enter the amount consumed in grams. The application automatically calculates total daily calories and macronutrients.

## Summary
This section displays graphs of daily nutrition intake, including calories, proteins, fats, and carbohydrates over time.

## Weight Tracking
In this tab, you can manually enter body weight. The application visualizes weight changes over time using graphs.

## How to use
After making changes in the application, click "Save file" to download and save the updated version to your computer. Next time, open the saved file to continue using the application with all previous data preserved. Without saving, all data will be lost after closing the file.
